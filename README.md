# 龙图阁 (Loong Tu Ge) - PDF Archive

This is a static website built with Vue 3 and Vite, designed to host and preview PDF documents on GitHub Pages.

## Features

- **File Explorer**: Browse files and directories in `public/Files`.
- **PDF Preview**: Preview PDF files directly in the browser using `vue-pdf-viewer`.
- **Search**: Search for files by name.
- **Responsive UI**: Clean interface with breadcrumb navigation.

## Document List

### Economics
- 财富、贫穷与政治.pdf

### Politics
- the-dictators-handbook.pdf

## Development

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## Deployment

The project is configured to automatically deploy to GitHub Pages via GitHub Actions when pushing to the `main` branch.
